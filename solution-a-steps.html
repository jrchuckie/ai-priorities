Brand Building 6步数据闭环

从 Attribute 定义 → 行业验证 → 品牌诊断 → 内容占领 → Brand Lift → 终端归因，首次打通品牌心智到销售的完整链路

00RESEARCH
Data Collection 收集主流 Research Firm 的 Attribute 体系
整合 Kantar、Nielsen、Ipsos、Millward Brown 等主流品牌研究机构的 brand attributes 标准框架，建立三维属性词库（功能性 / 情绪性 / 兴趣买点）。
Kantar BrandZ Nielsen Brand Health Ipsos Brand Value Creator Millward Brown
以 XHS UGC 大样本覆盖 Kantar 小样本调研盲区
01VALIDATE
First-Party Validation 用一方数据验证行业重要 Attributes
用品牌一方销售数据 × XHS UGC 内容打标，验证哪些 attributes 在该行业中真正影响消费者决策。去掉"看起来重要但不驱动转化"的属性，保留高 ROI attribute 清单。
UGC 内容打标 销售数据 × Attribute 相关性 行业 Attribute 重要性排序
实时验证，替代传统 6-8 周调研周期
02DIAGNOSE
Brand Imagery Diagnosis 诊断品牌当前 Attribute 占有率 vs 竞争对手
基于行业验证后的 attribute 清单，通过 XHS 站内 UGC 分析品牌在每个 attribute 的声量占有率（Share of Voice）与竞争水位——找出品牌优势位、缺失位、可进攻位。
Attribute SOV 竞品 Attribute 对比 品牌定位白地识别
03OCCUPY
Interest Ecosystem 圈定目标兴趣生态，精准内容投放
基于品牌战略选定的兴趣 Attributes（如赛车 / 露营 / 网球），在 XHS 站内对应兴趣圈层中做内容种草投放——让品牌出现在消费者最有共鸣的场景里。
Sponsorship 内容放大 兴趣圈层达人矩阵 场景化种草内容
Sponsorship 线上延伸，把线下 IP 声量转化为 XHS UGC
04MEASURE
Brand Lift 实时度量 Brand Lift — Attribute 认知提升
投放前后追踪每个 attribute 的 SOV 变化、情感倾向变化，输出品牌心智提升报告——以实时 UGC 数据替代 Kantar 季度调研，更快、更大样本、更便宜。
Attribute SOV 变化 情感倾向追踪 竞品 Benchmark
实时替代 Kantar 季度调研（更快 / 样本量 100x / 成本更低）
05ATTRIBUTE
Sales Attribution 终端销售归因 — 哪个 Attribute 驱动了购买
接入品牌一方销售数据（电商 / 线下 POS），构建品牌心智 → 终端销售归因模型。回答 CMO 最核心的问题：投了哪个兴趣 Sponsorship，ROI 最高？
一方数据回传 Attribute → 销售路径 Sponsorship ROI 排序
首个打通品牌心智 → 终端销售归因的品牌广告产品
